# 로그인 컨트롤러
`php artisan make:controller Auth/LoginController`
컨트롤러 생성 app/Http/Controllers/Auth/LoginController.php

- 반환할 폼을 등록함? <--- 이거부터 이미 신기함, 웹단 라우팅이 아니여도 여기서 처리가 된다고?

- $credentials / validate = 검증요청

    Auth:attempt 크레덴셜 정보 검증하는 조건문
        True: 세션생성, post로 api 메시지 return
        False:  에러메시지

# web.php에 라우트 등록

 middleware 개념: View의 posts 접근요청 < - > 미들웨어 < - >  컨트롤러의 요청 전달 사이에 기능을 실행
    로그인 인증을 요청해서 ->  없으면 컨트롤러에서 쳐냄


# 로그인 뷰로 접근
login.blade.php  생성, 당장은 로그인만 확인할거니 web.php 살짝 수정함

artisan migrate 확인 / serve로 개발용 서버 빌드
http://127.0.0.1:8000/login 접속하면 실행
