db 초기값 넣는 과정

database\seeders 에 초기값을 넣는 기능 지원

`php artisan make:seeder UserSeeder` 
해당 명령어로 생성 가능


근데 기존 코드 보니까 초기값부터 Hash화를 지원함 
Spring의 passwordEncoder.encode 라이브러리 기능과 유사

UserSeeder
User table 객체를 초기값으로 설정한 파일 

이걸 DatabaseSeeder.php에서 호출함 (다른 초기값이 있다면 DatabaseSeeder에 묶어서 컴파일 순으로 순차호출)

`php artisan migrate:fresh --seed`
마이그레이트 지우기 -> 재생성 -> seed값 읽어서 넣기
